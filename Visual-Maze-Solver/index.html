📖 How It Works

BFS uses a FIFO queue to explore cells level-by-level. Every cell at distance d is visited before any at distance d+1, guaranteeing the shortest path in an unweighted grid.

A* adds a heuristic h(n) = |Δrow| + |Δcol| (Manhattan distance) to guide the search toward the goal, dramatically reducing the number of cells explored while still finding the optimal path.
